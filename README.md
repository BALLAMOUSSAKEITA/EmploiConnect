# GestRH Guinée - Backend FastAPI

## Démarrage rapide

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

L'API sera disponible sur : http://localhost:8000
Documentation Swagger : http://localhost:8000/docs
